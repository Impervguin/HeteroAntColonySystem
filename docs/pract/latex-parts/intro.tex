\ssr{ВВЕДЕНИЕ}

Целью данной практики являются практическая реализация и исследование муравьиного алгоритма с динамическим подбором гетерогенных параметров на основе генетического алгоритма.

Были поставлены следующие задачи:

\begin{enumerate}
	\item реализовать на практике метод, разработанный в ходе выполнения выпускной квалификационной работы;
	\item описать и провести функциональное и модульное тестирование разработанного метода;
	\item исследовать характеристики реализованного программного обеспечения.
\end{enumerate}
